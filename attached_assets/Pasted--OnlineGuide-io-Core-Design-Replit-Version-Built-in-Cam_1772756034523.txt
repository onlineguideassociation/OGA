Built in 🇰🇭 Cambodia | Connected to 🌍 the World

1️⃣ Core App Architecture
Frontend (Web + Mobile)

Web App:

React.js + TailwindCSS + React Router

Key Features:

Knowledge Graph visualization (React Flow / D3.js)

Travel & cultural insights

Guide dashboards & media contributions

Marketplace access (commerce, media, souvenirs)

Personal traveler dashboards & eco-sustainability maps

Mobile App:

React Native (iOS/Android)

Shared components for cross-platform consistency

Backend

Node.js + Express APIs
Endpoints:
/api/temples | /api/guides | /api/travelers | /api/business | /api/farms | /api/properties | /api/media | /api/products

Database:

MongoDB / Firebase (cloud-hosted)

Stores: nodes & relationships, user profiles, transactions, media assets, contributions

AI Layer:

OpenAI GPT / Gemini 3 integration

Personalized travel recommendations, cultural storytelling, AI media tagging, commerce insights, revenue optimization

Deployment:

Initial: Replit (full-stack hosting)

Optional: AWS / Google Cloud / Vercel for scaling
